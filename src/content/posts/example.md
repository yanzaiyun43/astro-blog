## 什么是极简

极简不仅仅是"少即是多"（Less is more），它是一种**刻意选择**的结果...

## 核心原则

1. **功能性优先**：每个元素都应有其存在的理由
2. **留白的力量**：空白不是缺失，而是呼吸空间
3. **typographic 层次**：通过字体建立视觉秩序

```css
/* 极简配色示例 */
:root {
  --color-bg: #fafaf9;
  --color-text: #292524;
  --color-primary: #0ea5e9;
}
```

> "完美不是无可添加，而是无可删减。"
