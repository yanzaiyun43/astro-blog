## 博客缘起

在信息过载的时代，我渴望一个**干净、优雅、专注**的写作空间。旧识桥正是基于这样的理念诞生的——它不是又一个复杂的 CMS 系统，而是一个精心策划的数字花园，让内容回归本质。

旧识桥采用**静态站点生成**架构，将性能、美观与开发体验完美融合。作为 ailmel 的个人博客，这里记录我的技术学习、设计思考和生活感悟。

## 设计理念

### 极简主义美学

设计遵循"少即是多"的原则，但并非简单的元素删减：

- **柔和浅蓝**作为主色调，营造宁静的阅读氛围
- **低对比度文字**，减少视觉疲劳
- **充足的留白**，让内容自由呼吸
- **精致的微交互**，提升使用愉悦感

### 轻奢细节

在极简的基础上，我注入了一些精致的细节：

- 页面加载时的**淡入动画**
- 卡片悬停时的**微妙上浮效果**
- 平滑的**主题切换过渡**
- 精心调校的**字体层次**

## 核心特性

### 极速性能

基于 Astro 的**零 JavaScript 默认**架构，页面加载速度极快：

- 首屏渲染时间 < 100ms
- 完美的 Lighthouse 评分
- 静态 HTML 输出，CDN 友好

### 优雅的写作体验

```markdown
---
title: "我的文章"
description: "文章简介"
publishedAt: 2024-12-06
category: "分类"
tags: ["标签1", "标签2"]
featured: true
---

正文内容支持 Markdown 和 MDX...
```

### 完善的阅读功能

- **深色/浅色模式**自动切换
- **文章目录**自动生成
- **阅读进度**指示
- **相关文章**推荐
- **代码复制**功能

### 搜索引擎优化

内置完整的 SEO 支持：

- 自动生成 Open Graph 和 Twitter Card 元数据
- 结构化数据标记
- XML 站点地图
- RSS 订阅源

## 技术亮点

项目采用现代前端技术栈：

| 技术 | 用途 |
|------|------|
| [Astro](https://astro.build) | 静态站点生成器 |
| [Tailwind CSS](https://tailwindcss.com) | 实用优先的 CSS 框架 |
| [TypeScript](https://typescriptlang.org) | 类型安全的开发体验 |
| [Alpine.js](https://alpinejs.dev) | 轻量级 JavaScript 交互 |
